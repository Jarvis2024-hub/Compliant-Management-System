## Backend Requirements
- PHP 8.x (Tested on 8.2)
- MySQL 8.x (or MariaDB 10.x via XAMPP)
- XAMPP / WAMP (Recommended for local database management)
- Port 8000 (for PHP built-in server)

## Frontend Requirements (Flutter)
- Flutter SDK: >=3.0.0 <4.0.0
- Dart SDK: >=3.0.0 <4.0.0
- Android SDK (Min SDK 21, Target SDK 33+)
- Android Studio or VS Code

## Flutter Packages (Dependencies)
- http: ^1.1.0 (API Communication)
- google_sign_in: ^7.2.0 (Authentication)
- shared_preferences: ^2.2.2 (Local Storage)
- provider: ^6.1.1 (State Management)
- flutter_spinkit: ^5.2.0 (Loading Indicators)
- cupertino_icons: ^1.0.2 (Icons)

## Dev Dependencies
- flutter_launcher_icons: ^0.14.4
